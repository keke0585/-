# 视频研判辅助系统 (Smart Video Analysis System)

这是一个专为离线环境设计、基于深度学习内核的轻量级视频内容研判与目标提取工具。系统通过智能算法自动化识别、分析并提取录像中的关键目标，旨在大幅提升视频研判工作的效率。

## 项目概述

系统采用 B/S (Browser/Server) 架构，将复杂的深度学习后端与响应式 Web 前端相结合。用户可以轻松地在各类 Windows 环境中（包括隔离网络环境）部署并使用。通过集成先进的物体检测与追踪技术，系统能够从大规模视频素材中自动筛查指定目标，并生成结构化的分析结果。

## 核心功能

- **多模式检测方案**：基于 YOLO 系列模型提供人体、车辆以及人车双模检测，支持实时动态切换。
- **高精度人脸捕捉**：采用双模检测机制，优先使用 MediaPipe 获取高精度人脸，并提供 OpenCV Haar 级联作为兼容性兜底。
- **极速研判模式**：内置运动检测算法（MOG2）实现智能跳频，在静态画面下自动节省算力，在目标出现时即时触发高频采集。
- **智能化去重机制**：结合空间距离偏移与时间序列分析，有效过滤重复采集，确保抓拍结果的精准性。
- **多任务队列管理**：支持批量视频导入，提供自动化的分析队列管理及进度追踪功能。

## 技术架构

- **后端引擎**：采用 Python 3.8.10 开发，集成 Ultralytics 企业级检测框架及 PyTorch 运算库。
- **服务前端**：基于 Vanilla JavaScript 与 CSS 变量构建的轻量级 UI，不依赖外部框架，确保在离线断网环境下通过标准浏览器即可访问。
- **硬件适配层**：支持 Cross-Platform 算力调度，自动兼容 CUDA (Nvidia)、MPS (Apple Silicon) 以及标准 CPU 运算模式。
- **通信协议**：通过轻量级 REST API 进行状态同步，采用 MJPEG 流式传输方案实现低延迟的实时监控预览。

## 运行环境与部署

- **系统兼容性**：原生支持 Windows 10 与 Windows 11 环境。针对 Windows 7 及虚拟机隔离环境进行了底层优化，解决了 PyTorch 动态链接库加载等常见兼容性问题。
- **部署方式**：支持通过 PyInstaller 构建为独立的单文件可执行程序 (.exe)，无需在目标机器安装 Python 解释器或配置复杂的 AI 环境。
- **硬件要求**：推荐使用支持 CUDA 的 Nvidia 显卡以获得最佳性能，同时也支持在仅有 CPU 的入门级办公电脑上通过降级策略运行。

## 开发规范

本项目遵循高内聚、低耦合的设计原则，核心交互逻辑采用防御性编程模式，确保在复杂系统环境下具备极高的运行稳定性。
